# Giri-bilgiler-kodu
Bu proje, Python öğrenmeye başladıktan 6 gün sonra yazdığım temel giriş kontrol sistemi.
While döngüsü, if-else ve break kullanarak kullanıcı ve admin doğrulaması yapıyor.

## Kullanılan Konseptler
- input() ile veri alma
- if-elif-else ile koşul kontrolü
- while döngüsü ve break kullanımı

## Kullanım
1. Kod dosyasını çalıştırın.
2. Konsolda kullanıcı adı ve soyadı girin.
3. Giriş doğrulanırsa onay mesajı gösterilir.
